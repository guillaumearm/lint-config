# `@trapcodien/tsconfig`

My main tsconfig

## Usage

Install with `npm install --save-dev @trapcodien/tsconfig`


### with node

Create a `tsconfig.json` file with:

```json
{
  "extends": "@trapcodien/tsconfig"
}
```

### with react

Create a `tsconfig.json` file with:

```json
{
  "extends": "@trapcodien/tsconfig/tsconfig.react.json"
}
```
