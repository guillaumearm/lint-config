# `@trapcodien/tsconfig`

My main tsconfig

## Usage

### Installation

```shell
npm install -D typescript @trapcodien/tsconfig
```

### Basic configuration

Create a `tsconfig.json` file with:

```json
{
  "extends": "@trapcodien/tsconfig/tsconfig.base.json"
}
```

### React configuration:

```json
{
  "extends": "@trapcodien/tsconfig/tsconfig.react.json"
}
```
